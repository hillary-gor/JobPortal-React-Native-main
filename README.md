## Overview

JobFinder is a sleek and intuitive mobile job portal application built with React Native. It empowers job seekers to effortlessly discover and apply for their dream jobs, while offering employers a streamlined platform to post openings and manage applicants.

**Key Features:**

* **Intuitive Job Search:**  Browse popular and nearby jobs, filter by job type (part-time, full-time, contract), and utilize powerful search functionality.
* **Personalized Job Recommendations:**  Discover tailored job suggestions based on your preferences and profile.
* **Seamless Application Process:**  Upload resumes and cover letters directly through the app.
* **Employer Dashboard:**  (Optional) Enable employers to post job listings and review applications.

## Tech Stack

* **React Native:**  Framework for building native mobile apps.
* **Tailwind CSS:**  Utility-first CSS framework for rapid UI development.
* **React Native Navigation:**  Smooth navigation between app screens.
* **Redux:**  State management for predictable data flow.
* **Axios:**  Promise-based HTTP client for API interactions.
* **RapidAPI Jobs:** Job listing API (or your chosen API).

## Getting Started

1. **Clone the Repository:**
   ```bash
   git clone [https://github.com/](https://github.com/)
   ```

2. **Install Dependencies:**

   ```bash
   cd JobPortal-React-Native
   npx expo install
   ```

3. **Start the App:**

   ```bash
   npx expo start
   ```

## Deployment (Optional)

**Expo Publish:**

```bash
expo publish
```
